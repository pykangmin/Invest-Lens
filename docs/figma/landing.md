출처: `images/진입 화면.png` (3.18MB, 가장 큰 시안 파일이라 판독률 높음)

---

## 화면 목적

종목 검색 진입점. 사용자가 종목명을 타이핑하면 자동완성 또는 엔터로 개별 주식 대시보드로 이동.

---

## 전체 구도 ✅

- 풀블리드 다크 hero 화면. 데스크톱 우선 레이아웃
- 가로:세로 ≈ 1.36:1 (시안 PNG 597×440 기준 — 실 디자인 캔버스는 16:9 추정)
- 콘텐츠는 화면 중앙 단일 컬럼에 수직 정렬
- 스크롤 없음. 한 화면 안에 모든 요소 들어감

수직 순서 (위 → 아래):

1. 로고
2. 헤드라인 (메인 카피)
3. 부제 (서브 카피)
4. 검색 입력
5. 인기 검색어 라인

---

## 배경 ✅ + ⚠️

- ✅ **베이스 컬러**: 깊은 다크 네이비. 시안 픽셀 추정 hex `#0B1E3F` 근처 (⚠️ 정확치는 디자이너 확인)
- ✅ **텍스처**: 캔들 차트 + 라인 차트가 반투명으로 화면 전반에 깔림
  - 캔들: 빨강·녹색 (상승/하락 의미 색)
  - 라인: 옅은 청록·회백
  - 큰 가격 숫자 일부가 배경에 박혀 있음 — 판독되는 것: `38,371` `46,060` `43,604` `48,257` `192,201` `10,623` (이건 mock 일 가능성)
- ⚠️ **opacity**: 차트 패턴은 0.15 ~ 0.25 추정. hero 텍스트 영역(중앙)은 패턴이 약간 더 흐릿해 가독성 확보
- ⚠️ **그라데이션**: 중심부가 약간 더 어두워 vignette 반대 효과를 연출. 명확치 않음

→ 21-aesthetics 의 "배경" 정책에 부합하는지 확인 후 적용. 만약 21-aesthetics 가 단색 배경을 강제한다면 차트 패턴 사용은 21-aesthetics 본문 수정 합의 후 진행.

## 로고 ✅

- 마크: `〉` (chevron right 단일 표시)
- 워드: `Invest Lens`
- 워드 폰트: ⚠️ 이탤릭 시리프 또는 시리프 톤의 산세리프 (Playfair Display / Cormorant 류 추정)
- 색: 흰색
- 마크 색: ⚠️ 워드와 같은 흰색일 수도, 약간 다른 amber/gold 톤일 수도. 정확치 불명
- 위치: 화면 상단 중앙. 헤드라인 위 약간 떨어진 곳
- 크기: 헤드라인의 약 1/4 ~ 1/3

## 헤드라인 ✅

원문 (2줄):

```
투자의 시각,
데이터로 투명하게
```

- 줄바꿈: 콤마 뒤에서 한 번
- 정렬: 가운데
- 색: 흰색
- ⚠️ **강조어**: "투명" 한 단어만 약간 다른 색으로 처리된 것으로 보임
  - 추정: 옅은 amber/gold (`#E0B556` 추정) 또는 약간 더 밝은 톤
  - 강조 폭: "투명" 만 (조사 "하게" 는 흰색)
- 폰트: 굵은 한글 산세리프 (Pretendard Bold/Black 추정)
- 크기: ⚠️ 시안 비율로 56~64px h1 추정
- 행간: 1.15 추정
- 자간: 약간 좁음 (-0.02em 추정)

## 부제 ✅ + ⚠️

원문:

```
종목명을 입력하면 펀더멘탈부터 ___까지 한눈에 분석해드립니다.
```

- ⚠️ `___` 는 시안에서도 빈칸. 어떤 단어가 들어갈지 미정
  - 후보: "거시 환경" / "기술적 신호" / "투자 인사이트" / "리스크 시그널"
  - 결정 필요. 02-data-analysis 가 다루는 4개 영역(펀더멘탈·기술적·거시·원자재) 또는 03-insight 의 영역명을 그대로 가져오는 안 권장
- 정렬: 가운데
- 색: 옅은 회색 (~`#9BA8BD` 추정), 흰색보다 한 단계 낮은 명도
- 폰트: 한글 산세리프 (Pretendard Regular 추정)
- 크기: ⚠️ 14~16px

## 검색바 ✅

- 형태: pill (양 끝이 완전 둥근 원형 종단)
- 가로 폭: 화면의 ~50% (헤드라인보다 약간 넓음)
- 높이: ⚠️ 48~56px 추정
- 배경: 거의 흰색에 가까운 매우 옅은 컬러. 약간 투명/블러
- 좌측: 돋보기 아이콘 (옅은 회색)
- placeholder: `주식 종목을 입력하세요`
- placeholder 색: 회색 (~`#A0AEC0` 추정)
- 우측 끝: 원형 submit 버튼
  - 형태: 완전 원형
  - 배경: 매우 어두운 인디고 또는 차콜 (`#0F1A33` 추정)
  - 아이콘: 흰색 화살표 `→`
  - 크기: 검색바 높이의 약 75%, 검색바 우측 padding 안에 fit

## 인기 검색어 라인 ✅

레이아웃: `라벨 + 칩 4개` 가 한 줄에 가운데 정렬

라벨:
- ⚠️ 이모지: 👍 (옅은 amber/peach 톤)
- 텍스트: `인기 검색어`
- 색: 흰색
- 크기: 부제와 비슷하거나 한 단계 작음

칩 4종 (좌→우):

1. `삼성 전자` (스페이스 포함)
2. `GOOGLE`
3. `MSFT`
4. `NVDA`

칩 스타일:
- 형태: pill (라운드 풀)
- 보더: 1px solid 흰색 또는 옅은 회색 (~`#FFFFFF20` 또는 `#9BA8BD`)
- 배경: 투명 또는 매우 옅은 다크 (~`#FFFFFF08`)
- 글자: 흰색, 작은 사이즈
- 패딩: 좌우 ~14px, 상하 ~6px (추정)
- 칩 간 간격: ~12px

⚠️ **데이터 합의 필요**: `삼성 전자` 는 한국 종목이지만 현재 `company_master` 는 S&P 500 503개만. 이 칩을 누르면 종목이 안 나옴. 옵션:
- (a) 칩 라벨을 `AAPL` 등 S&P 종목으로 갈아끼움
- (b) 한국 종목 별도 조달
- (c) "인기 검색어" 를 S&P 500 의 거래량/시총 TOP 으로 동적 산출
권장: (c) — 시안의 4종은 mock 으로 보고, 실제 구현은 동적 TOP. 시안 충실도와 데이터 일관성을 둘 다 잡음.

## 인터랙션 ❓

시안은 정적 PNG 라 인터랙션은 알 수 없음. 추정:

- 검색 입력 → 자동완성 드롭다운 (등장 위치/스타일 미상)
- 인기 검색어 칩 클릭 → 즉시 검색 실행 (대시보드로 이동)
- submit 버튼 → 입력 내용으로 검색 실행
- hover 효과: 칩에 약한 배경 채움? ❓
